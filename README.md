# PRODIGY_DS_01 — Data Distribution Visualization

**Task 01 of the Prodigy InfoTech Data Science Internship**

## 📌 Task Description

Create a bar chart or histogram to visualize the distribution of a categorical or continuous variable, such as the distribution of ages or genders in a population.

## 📊 Overview

This project performs exploratory data analysis (EDA) on a dataset and visualizes the distribution of a chosen variable using bar charts and/or histograms. The goal is to uncover patterns in the data — such as how a population is distributed across categories (e.g., gender) or across a numerical range (e.g., age) — through clear, well-labeled visualizations.
